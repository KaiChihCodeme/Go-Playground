# Multiple-go-version-in-one-machine
## Install diff go version to one computer
`go install golang.org/dl/go1.22.1@latest`
`go1.22.1 download`
`go1.22.1 version`
`go1.22.1 env GOROOT`
`alias go="go1.22.1"`
then `go version`
you will see the version change to you specify.

https://go.dev/doc/manage-install

## Change go version
`alias go="go1.22.1"`
